InstructorTips
version 0.57
Kevin Santer (ksanter@michiganvirtual.org)
InstructorTips is a tool for building schedules with checklists of weekly tips.
  • create schedules, specifying the number of weeks and the start date
  • write their own tips and add them to any schedule
  • reuse tips on any schedule
  • choose from a repository of common tips
  • classify tips e.g. course launch, engagement, course end, etc.
  • share tips and schedules with other users
  • enable weekly notification reminders of tips that have yet to be checked off
